## 🔧 工具类使用

### ResultUtils 工具类
```java
// 成功返回
ResultUtils.success(data)
ResultUtils.success()

// 错误返回
ResultUtils.operationFailed("操作失败")
ResultUtils.dataNotFound("数据不存在")
ResultUtils.stockInsufficient()
ResultUtils.paramError("参数错误")
```

### ResultCode 枚举
```java
ResultCode.OPERATION_FAILED      // 操作失败
ResultCode.DATA_NOT_FOUND        // 数据不存在
ResultCode.CANNOT_DELETE         // 无法删除（存在关联数据）
ResultCode.STOCK_INSUFFICIENT    // 库存不足
ResultCode.ORDER_STATUS_ERROR    // 订单状态异常
```

## 🎯 改进原则

1. **统一错误码**：使用 `ResultCode` 枚举，不使用硬编码
2. **统一异常处理**：业务异常使用 `BusinessException`，让全局处理器处理
3. **完善日志**：关键操作记录日志，使用合适的日志级别
4. **空指针防护**：关键数据必须进行 null 检查
5. **代码简洁**：减少不必要的 try-catch，让代码更清晰
