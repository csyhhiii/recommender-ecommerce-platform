# 代码质量改进示例

本文档展示如何使用新的工具类改进代码质量。

---

## 一、使用 ResultUtils 工具类

### ❌ 改进前

```java
public Result<?> createProduct(Product product) {
    try {
        int result = productMapper.insert(product);
        if (result > 0) {
            LOGGER.info("创建商品成功，商品ID：{}", product.getId());
            return Result.success(product);
        }
        return Result.error("-1", "创建商品失败");
    } catch (Exception e) {
        LOGGER.error("创建商品失败：{}", e.getMessage());
        return Result.error("-1", "创建商品失败：" + e.getMessage());
    }
}
```

### ✅ 改进后

```java
public Result<?> createProduct(Product product) {
    try {
        int result = productMapper.insert(product);
        if (result > 0) {
            LOGGER.info("创建商品成功，商品ID：{}", product.getId());
            return ResultUtils.success(product);
        }
        return ResultUtils.operationFailed("创建商品失败");
    } catch (Exception e) {
        LOGGER.error("创建商品失败：{}", e.getMessage(), e);
        throw new BusinessException(ResultCode.OPERATION_FAILED, "创建商品失败：" + e.getMessage());
    }
}
```

## 二、使用 ResultCode 枚举

### ❌ 改进前

```java

public Result<?> getProductById(Long id) {
```

## 三、优化异常处理

### ❌ 改进前（捕获所有异常）

```java
public Result<?> updateProduct(Long id, Product product) {
    try {
        // 检查库存是否合法
        if (product.getStock() < 0) {
            return Result.error("-1", "库存不能为负数");
        }
        
        int result = productMapper.updateById(product);
        if (result > 0) {
            LOGGER.info("更新商品成功，商品ID：{}", id);
            return Result.success(product);
        }
        return Result.error("-1", "更新商品失败");
    } catch (Exception e) {
        LOGGER.error("更新商品失败：{}", e.getMessage());
        return Result.error("-1", "更新商品失败：" + e.getMessage());
    }
}
```

### ✅ 改进后（使用 BusinessException）

```java
public Result<?> updateProduct(Long id, Product product) {
    // 检查库存是否合法
    if (product.getStock() < 0) {
        throw new BusinessException(ResultCode.PARAM_ERROR, "库存不能为负数");
    }
    
    product.setId(id);
    int result = productMapper.updateById(product);
    if (result > 0) {
        LOGGER.info("更新商品成功，商品ID：{}", id);
        return ResultUtils.success(product);
    }
    throw new BusinessException(ResultCode.OPERATION_FAILED, "更新商品失败");
    // 异常会被 GlobalExceptionHandler 统一处理
}
```

## 五、完善日志记录

### ❌ 改进前（日志不完整）

```java
public Result<?> deleteProduct(Long id) {
    int result = productMapper.deleteById(id);
    if (result > 0) {
        return Result.success();
    }
    return Result.error("-1", "删除失败");
}
```

### ✅ 改进后（完善日志）

```java
public Result<?> deleteProduct(Long id) {
    try {
        // 检查关联数据
        if (hasRelatedData(id)) {
            LOGGER.warn("尝试删除商品ID：{}，但存在关联数据", id);
            return ResultUtils.error(ResultCode.CANNOT_DELETE, "无法删除，存在关联数据");
        }
        
        int result = productMapper.deleteById(id);
        if (result > 0) {
            LOGGER.info("删除商品成功，商品ID：{}", id);
            return ResultUtils.success();
        }
        LOGGER.warn("删除商品失败，商品ID：{}，可能不存在", id);
        return ResultUtils.dataNotFound("商品不存在");
    } catch (Exception e) {
        LOGGER.error("删除商品异常，商品ID：{}，错误：{}", id, e.getMessage(), e);
        throw new BusinessException(ResultCode.OPERATION_FAILED, "删除商品失败");
    }
}
```

## 六、空指针防护

### ❌ 改进前（缺少 null 检查）

```java
public Result<?> getProductById(Long id) {
    Product product = productMapper.selectById(id);
    return Result.success(product); // 可能返回 null
}
```

### ✅ 改进后（添加 null 检查）

```java
public Result<?> getProductById(Long id) {
    Product product = productMapper.selectById(id);
    if (product == null) {
        LOGGER.warn("查询商品失败，商品ID：{} 不存在", id);
        return ResultUtils.dataNotFound("商品不存在");
    }
    return ResultUtils.success(product);
}
```

## 七、完整示例：改进后的 Service 方法

```java
@Service
public class ProductService {
    private static final Logger LOGGER = LoggerFactory.getLogger(ProductService.class);
    
    @Autowired
    private ProductMapper productMapper;
    
    /**
     * 创建商品
     */
    public Result<?> createProduct(Product product) {
        try {
            // 参数验证（已在 Controller 层通过 @Valid 验证）
            
            int result = productMapper.insert(product);
            if (result > 0) {
                LOGGER.info("创建商品成功，商品ID：{}，商品名称：{}", product.getId(), product.getName());
                return ResultUtils.success(product);
            }
            LOGGER.warn("创建商品失败，数据库插入返回 0");
            throw new BusinessException(ResultCode.OPERATION_FAILED, "创建商品失败");
        } catch (BusinessException e) {
            throw e; // 重新抛出业务异常
        } catch (Exception e) {
            LOGGER.error("创建商品异常：{}", e.getMessage(), e);
            throw new BusinessException(ResultCode.OPERATION_FAILED, "创建商品失败：" + e.getMessage());
        }
    }
    
    /**
     * 根据ID查询商品
     */
    public Result<?> getProductById(Long id) {
        Product product = productMapper.selectById(id);
        if (product == null) {
            LOGGER.warn("查询商品失败，商品ID：{} 不存在", id);
            return ResultUtils.dataNotFound("商品不存在");
        }
        return ResultUtils.success(product);
    }
    
    /**
     * 更新商品
     */
    public Result<?> updateProduct(Long id, Product product) {
        // 检查商品是否存在
        Product existingProduct = productMapper.selectById(id);
        if (existingProduct == null) {
            LOGGER.warn("更新商品失败，商品ID：{} 不存在", id);
            return ResultUtils.dataNotFound("商品不存在");
        }
        
        // 业务验证
        if (product.getStock() != null && product.getStock() < 0) {
            throw new BusinessException(ResultCode.PARAM_ERROR, "库存不能为负数");
        }
        
        product.setId(id);
        int result = productMapper.updateById(product);
        if (result > 0) {
            LOGGER.info("更新商品成功，商品ID：{}", id);
            return ResultUtils.success(product);
        }
        throw new BusinessException(ResultCode.OPERATION_FAILED, "更新商品失败");
    }
    
    /**
     * 删除商品
     */
    public Result<?> deleteProduct(Long id) {
        // 检查商品是否存在
        Product product = productMapper.selectById(id);
        if (product == null) {
            LOGGER.warn("删除商品失败，商品ID：{} 不存在", id);
            return ResultUtils.dataNotFound("商品不存在");
        }
        
        // 检查关联数据（示例）
        // if (hasRelatedOrders(id)) {
        //     return ResultUtils.error(ResultCode.CANNOT_DELETE, "无法删除，存在关联订单");
        // }
        
        int result = productMapper.deleteById(id);
        if (result > 0) {
            LOGGER.info("删除商品成功，商品ID：{}", id);
            return ResultUtils.success();
        }
        throw new BusinessException(ResultCode.OPERATION_FAILED, "删除商品失败");
    }
}
```

## 八、改进要点总结

1. **使用 ResultUtils**：简化 Result 的创建，统一错误码
2. **使用 ResultCode 枚举**：避免硬编码错误码
3. **使用 BusinessException**：让全局异常处理器统一处理
4. **完善日志记录**：关键操作记录日志，使用合适的日志级别
5. **空指针检查**：关键数据必须进行 null 检查
6. **减少 try-catch**：让全局异常处理器处理异常，代码更简洁
