## 📐 子图3：后端API网关与认证流程

```mermaid
sequenceDiagram
    participant Client as 客户端<br/>Vue.js
    participant Gateway as API网关<br/>/api
    participant JWT as JWT拦截器<br/>JwtInterceptor
    participant Controller as Controller层
    participant Service as Service层
    participant Cache as 缓存层<br/>Redis/Caffeine
    participant DB as MySQL数据库
    participant External as 外部服务
    
    Note over Client,External: 认证流程
    Client->>Gateway: HTTP请求<br/>/api/user/login
    Gateway->>JWT: 检查是否需拦截
    JWT->>JWT: 排除登录接口
    JWT->>Controller: 放行请求
    Controller->>Service: 调用业务逻辑
    Service->>DB: 验证用户信息
    DB-->>Service: 返回用户数据
    Service->>Service: 生成JWT Token
    Service-->>Controller: 返回Token
    Controller-->>Client: 返回登录结果+Token
    
    Note over Client,External: 业务请求流程（需认证）
    Client->>Gateway: HTTP请求<br/>/api/product/list<br/>Header: token=xxx
    Gateway->>JWT: 检查Token
    JWT->>JWT: 解析Token
    JWT->>Service: 验证用户存在
    Service->>DB: 查询用户信息
    DB-->>Service: 用户数据
    Service-->>JWT: 用户有效
    JWT->>JWT: 验证Token签名
    alt Token有效
        JWT->>Controller: 放行请求
        Controller->>Service: 调用业务逻辑
        Service->>Cache: 查询缓存
        alt 缓存命中
            Cache-->>Service: 返回缓存数据
        else 缓存未命中
            Service->>DB: 查询数据库
            DB-->>Service: 返回数据
            Service->>Cache: 写入缓存
        end
        Service-->>Controller: 返回业务数据
        Controller-->>Client: 返回结果
    else Token无效
        JWT-->>Client: 401 Unauthorized<br/>Token失效
    end
    
    Note over Client,External: 外部服务调用（支付）
    Client->>Gateway: 创建订单<br/>/api/order/create
    Gateway->>JWT: 验证Token
    JWT->>Controller: 放行
    Controller->>Service: 创建订单
    Service->>DB: 保存订单
    Service->>External: 调用支付宝API<br/>/alipay/create
    External-->>Service: 返回支付页面
    Service-->>Controller: 支付页面HTML
    Controller-->>Client: 返回支付页面
```

## 📋 架构说明

### 1. 系统架构特点

- **前后端分离**：Vue.js前端 + Spring Boot后端，通过RESTful API通信
- **JWT认证**：基于Token的无状态认证机制
- **多层缓存**：Caffeine本地缓存 + Redis分布式缓存
- **外部服务集成**：支付宝支付、邮件服务、AI客服等

### 2. 技术栈

**前端**：
- Vue.js 3
- Vue Router（路由管理）
- Vuex（状态管理）
- Axios（HTTP客户端）

- Spring Security（安全框架）

**外部服务**：

### 3. 核心功能模块

- **用户管理**：注册、登录、个人信息管理

```mermaid
sequenceDiagram
    participant User as 用户
    participant Frontend as 前端<br/>Cart.vue
    participant CartController as CartController
    participant OrderController as OrderController
    participant OrderService as OrderService

    User->>Frontend: 选择商品，点击结算
    Frontend->>Frontend: 验证收货地址
    Frontend->>OrderController: POST /api/order/cart/create<br/>{cartIds, userId, 地址信息}
    OrderController->>OrderService: createOrdersFromCart()
    
    OrderService->>CartService: 查询购物车商品
    CartService->>MySQL: 查询cart表
    MySQL-->>CartService: 返回购物车数据
    CartService-->>OrderService: 购物车商品列表
    
    loop 遍历每个购物车项
        OrderService->>ProductService: 查询商品信息
        ProductService->>Redis: 查询商品缓存
        alt 缓存命中
            Redis-->>ProductService: 返回商品信息
        else 缓存未命中
            ProductService->>MySQL: 查询product表
            MySQL-->>ProductService: 商品数据
            ProductService->>Redis: 写入缓存
        end
        ProductService-->>OrderService: 商品信息
        
        OrderService->>OrderService: 检查库存
        alt 库存充足
            OrderService->>MySQL: 创建订单记录
            MySQL-->>OrderService: 订单ID
        else 库存不足
            OrderService-->>OrderController: 返回错误：库存不足
            OrderController-->>Frontend: 错误提示
            Frontend-->>User: 提示库存不足，终止流程
        end
    end
    
    OrderService->>MySQL: 生成商户订单号(outTradeNo)
    OrderService-->>OrderController: 返回订单创建成功<br/>{outTradeNo, orderCount}
    OrderController-->>Frontend: 订单创建成功
    
    Frontend->>AlipayController: POST /alipay/create<br/>{orderNo, orderName, payPrice}
    AlipayController->>AlipayService: create()创建支付订单
    AlipayService->>Alipay: 调用支付宝API<br/>创建支付订单
    Alipay-->>AlipayService: 返回支付页面HTML
    AlipayService-->>AlipayController: 支付页面HTML
    AlipayController-->>Frontend: 返回支付页面
    Frontend-->>User: 显示支付页面
    
    User->>Alipay: 完成支付
    Alipay->>AlipayController: 支付回调<br/>POST /alipay/notify
    AlipayController->>AlipayService: 验证支付结果
    AlipayService->>Alipay: 验证签名
    Alipay-->>AlipayService: 验证通过
    
    AlipayService->>OrderService: updateOrderPaymentStatus()<br/>更新订单状态
    loop 更新所有相关订单
        OrderService->>MySQL: 更新订单状态为已支付
        OrderService->>ProductService: 扣减库存
        ProductService->>MySQL: 更新商品库存和销量
        OrderService->>CartService: 清理购物车
        CartService->>MySQL: 删除购物车记录
    end
    
    OrderService->>Redis: 清除相关缓存
    OrderService-->>AlipayService: 订单更新成功
    AlipayService-->>AlipayController: 支付成功
    AlipayController-->>Alipay: 返回success
    AlipayController-->>Frontend: 支付成功通知
    Frontend-->>User: 显示支付成功，跳转订单页面

    UserService-->>JWTInterceptor: 用户存在
    JWTInterceptor->>UserController: 放行请求
    UserController->>UserService: 查询用户信息
    UserService->>MySQL: 查询用户详情
    MySQL-->>UserService: 用户数据
    UserService-->>UserController: 返回用户信息
    UserController-->>Frontend: 返回用户数据
    Frontend-->>User: 显示用户信息
```
