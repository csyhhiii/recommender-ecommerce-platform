## 二、技术实现细节

### 前端技术栈
- Vue 3 + Element Plus
- 组件化开发
- 响应式设计
- CSS3动画效果

### 后端接口使用
- `/recommend/user/{userId}/explanation` - 获取带解释的推荐
- `/recommend/user/{userId}` - 普通推荐（降级使用）
- `/favorite/user/{userId}` - 获取收藏状态

### 数据结构
推荐响应数据结构：
```json
{
  "products": [...],
  "algorithmType": "COLLABORATIVE",
  "explanations": [
    {
      "reasonType": "COLLABORATIVE",
      "reasonDescription": "推荐理由描述",
      "score": 8.5,
      "confidence": 0.92,
      "similarUserCount": 156
    }
  ],
  "stats": {
    "accuracy": 0.92,
    "clickRate": 0.152,
    "executionTime": 8700
  }
}
```

### 本地存储
- `algorithmWeights` - 保存算法权重配置
- `frontUser` - 用户信息
- `token` - 认证令牌

本次优化成功将推荐算法从"看不见的后台"转变为"可感知、可交互、可解释"的前台功能。通过以下方式实现了核心目标：

4. **算法可感知** - 通过视觉设计和交互设计突出算法价值
